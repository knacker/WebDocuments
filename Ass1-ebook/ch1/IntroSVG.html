Creating a SVG

Next, we are going to talk about what a SVG is and discuss the elements, that are going to be used for our example. So what is a SVG actually? A SVG is a Scalable Vector graphic, that is recommended, like MathML, by the W3C. SVG's have a few advantages over regular png's, such as smaller size aswell as the ability, to scale with the screen size. In the following table you can see, what elements of SVG can be used to create a coordinate system and a polygon.

Elements that are going to be used

<element> Description
<polygon> The polygon element is being used to create an object, that contains atleast
three sides. It is made of straight lines and the shape is closed
<line> This can be used to create a simple line, that connects 2 points.
<text>
 Using the <text> element, we can write simple text in the SVG namespace.
