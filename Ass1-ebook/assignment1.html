1 Introduction to the XML Mind ebook Format

Introducing ebook

XML Mind ebook is a set of HTML pages turned into a self-contained ebook. We are going to use it, to create an ebook using our HTML files, that we are going to create in the next paragraps. Using these, we will have a small example for an ebook.

1.1 Using MathML in ebook

Using MathML

First of all, I will tell you what MathML actually is. MathML is a document format, that is being used to express mathematical formulas and complex expressions. Since 2010, it is recommended by W3C. In the following table, I will introduce the elements, that are needed for creating our formula, which I will talk about more in depth soon.

Elements that are going to be used

<element>Description
<mtext>This is being used to write text inside the math namespace.
<mrow>The Term <mrow> is being used to group expressions together .
<mi>This can be used for variable names, such as x or other Symbols.
<mo>We can use this to write down Symbols, such as ",".
<mfrac>Using this, will create a fraction, where the first argument will be the numerator
and the second argument will be the denominator.

1.2 Using SVG in ebook

Creating a SVG

Next, we are going to talk about what a SVG is and discuss the elements, that are going to be used for our example. So what is a SVG actually? A SVG is a Scalable Vector graphic, that is recommended, like MathML, by the W3C. SVG's have a few advantages over regular png's, such as smaller size aswell as the ability, to scale with the screen size. In the following table you can see, what elements of SVG can be used to create a coordinate system and a polygon.

Elements that are going to be used

<element>Description
<polygon>The polygon element is being used to create an object, that contains atleast
three sides. It is made of straight lines and the shape is closed
<line>This can be used to create a simple line, that connects 2 points.
<text>
Using the <text> element, we can write simple text in the SVG namespace.

2.1 How to Express the Formula with MathML

How the formula is created

To represent the formula, we must first consider the T on the left side of the equation. This can be represented very simply by the <mi> T </mi>. For the "=" we now also use a relatively simple <mo> = </mo>. Now comes the middle part of the equation, which is the difficult part. First of all we have to use <mfrac> to represent a fraction. Here we expect a numerator and denominator. The numbers are now defined with <mi> 1 </mi> and <mi> 2 </mi>. Next, we set the amount lines, which will be set again later in the same way after closing the brackets. To place the brackets around the matrix correctly, <mrow> must be used. This allows a bracket replacement that covers the entire matrix. Finally, the matrix is closed again with brackets. To display the matrix, the <mtable> element is used. Thereby <mtr> forms a new row entry. The indices are created by using <msub>. The first entry is the variable and the second entry is the index. In the third part of the equation, we use practically the same parts as in the middle part, except that a simple subtraction is sufficient and the use of <msub>. To insert our text now into our ebook, we have to use the namespace of MathML in our chapter for the correct display of the formula. Everything within the <math> tag will be recognized accordingly. 2.2 How to Draw the Triangle with SVG

Simple steps to create an SVG

A triangle with SVG can be drawn using the polygon element. <polygon points="50,50 60,10 85,45">. The values shown are the tuples(50,50) (60,10) (85,45) stand for the coordinates, where each point of the triangle is located in the grid. Using style="stroke:rgb(0,0,0);stroke-width:2" you can set the desired style you want to use for your triangle. In this particular example, the fill of the triangle is black, which is represented by the rgb value (0,0,0). The line thickness can be set using stroke-width, which is set here to 2.
